Plugins
tpm -> Tmux plugin manager (just use it)
* Makes installing plugins really easy

tmux-ressurect
* Saves and restores tmux sessions even when the server is killed.
Works similarly to vim sessions

tmux-ressurect-strategy-nvim
* Saves and restores Neovim sessions through vim-obsession
* By default, tmux does not restore saved vim sessions

tmux-yank
* Gives you copy-paste to system clipboard functionality in tmux.
* Particularly useful in vim-mode
